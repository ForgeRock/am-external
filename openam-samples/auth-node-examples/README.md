<b>Authentication Node Examples</b>
<br/>
This folder contains a range of community created authentication nodes, for use with the Authentication Trees framework.  Each node has a separate README.md explaining compilation, installation and usage.  These nodes are "PoC" quality for use as testing, examples and demo-ware.  For a list of current node development plans see https://wikis.forgerock.org/confluence/pages/viewpage.action?spaceKey=AMPLAN&title=Authentication+Node+Planning.  For further information email simon.moffatt@forgerock.com
