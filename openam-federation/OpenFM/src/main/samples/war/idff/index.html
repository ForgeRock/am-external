Introduction

In this sample, we demonstrate the different ID-FF protocols such as Account Federation, Single Sign On, Single Logout and Federation Termination.

Setup

You need two Open Federation instances to setup and run this sample. One of them as an Identity Provider; and the other as a Service Provider.

You must run the Configurator for the two instances first before configuring this sample.

To configure this instance as an Identity Provider; click here.

To configure this instance as a Service Provider, click here.

Demonstration

It is recommended that you follow the instructions in the FAQ about the IDFF sample to demonstate the various features.

If you have setup this instance as a Service Provider, click here to try out the different protocols that the Service Provider supports.
